# Personal Portfolio

A modern personal portfolio built with React, TypeScript, TanStack Router, and MDX. Features elegant design with smooth animations, dark/light theme, and a blog system.

## Features

- **Dynamic routing** with TanStack Router
- **Blog system** with MDX content
- **Dark/light theme** with persistence
- **SEO optimization** built-in
- **Reusable components** with Shadcn UI
- **Visual animations and effects** (canvas particles, gradients, morphing blobs)
- **Functional contact form**
- **Fully responsive** and accessible

## Sections

- **Home** - Hero section with introduction
- **About** - Personal and professional description
- **Projects** - Featured project portfolio
- **Stack** - Technologies and tools used
- **Experience** - Professional experience timeline
- **Blog** - MDX articles
- **Contact** - Contact form

## Tech Stack

- **React 18** with TypeScript
- **Vite** for build and development
- **TanStack Router** for routing
- **TailwindCSS** for styling
- **Shadcn UI** for base components
- **MDX** for blog content
- **Zod** for validation

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
